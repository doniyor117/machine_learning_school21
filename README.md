# Machine Learning Fundamentals

A personal archive of my machine learning studies and experiments
These are not optimized for production, they are built to help grasp the underlying calculus and linear algebra before moving on to tools like Scikit-Learn and TensorFlow."

Developed as part of the **School 21 (Ecole 42)** Data Science curriculum.

| Project Directory | Description & Key Concepts |
|-------------------|----------------------------|
| **`ml1_introduction`** | **Data Analysis & EDA**: Introduction to data manipulation using Pandas/NumPy, statistical analysis, and data visualization. |

## üöÄ How to Run

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/doniyor117/machine_learning_school21.git](https://github.com/doniyor117/machine_learning_school21.git)
    cd machine_learning_school21
    ```

2.  **Navigate to a project folder (e.g., Classification):**
    ```bash
    cd ml4_classification
    ```

3.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Launch the Notebook:**
    ```bash
    jupyter notebook ml4_classification.ipynb
    ```
